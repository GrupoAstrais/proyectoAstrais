# Proyecto final: Astrais

![Captura de pantalla](./docs/images/banner.svg)

Astrais ayuda a la gamificación de hábitos y tareas cotidianas. Astraïs convierte la productividad en un “progreso de personaje”. Más allá de gestionar tareas, el usuario puede notar crecimiento continuo: sube de nivel, desbloquea cosméticos, personaliza su avatar, adopta mascotas y accede a minijuegos.
